実績解除ったーは以下のTwitter権限を要求します。

  • アカウントやタイムラインの読み取り権限
  • 投稿権限

このうち読み取り権限については一切使用しません。 Twitter連携アプリの仕様上、画像を投稿するだけでも読み取り権限も同時に必要になります。

また、Twitterとの通信は独自の中継サーバを通して行われるため、キーが漏洩して乗っ取られることはありません。 この中継サーバは何もデータを保存しないため、あなたの認証情報が保存されることもありません。 中継サーバのソースコードは GitHubで公開されている ので、何も保存していないことを確認できます。

「実績解除ったー」の認証情報はあなたのPC/スマートフォンに保存されます。 アプリの認証情報だけではアカウント乗っ取り等はできませんが、 認証情報が盗まれると「実績解除ったー」を通した投稿だけは可能になってしまいます。 公共のPCなどで利用する際は、使い終わったら必ず「ログアウト」してください。
